title: 'Velox Vectors'
description: 'Intro to Velox Vectors'
pubDate: 'Feb 01 2026'
---

Velox vectors are in-memory representation of columnar data in Velox and are used as input and output to various components in Velox.
Designed for vectorized execution model, Velox vecor extend Apache Arrow Format.

All Vector data is stored in Velox Buffer, which is contigous memory allocation from memory Pool.
